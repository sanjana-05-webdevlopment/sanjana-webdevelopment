About Us

We are passionate learners and developers who love building simple and beautiful web pages using HTML and CSS.

Our goal is to make learning web design fun, practical, and beginner-friendly.

  • We use semantic HTML to improve accessibility
  • CSS is used for layout, colors, and style
  • Responsive design helps pages look good on all devices

By learning and applying these simple techniques, you can build your own personal or project websites from scratch!
